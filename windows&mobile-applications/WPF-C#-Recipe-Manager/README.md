# Recipe Application GUI

This project is a WPF application that allows users to manage recipes. The application supports entering new recipes, displaying all recipes, selecting and displaying a specific recipe, resetting and scaling recipe quantities, setting high-calorie notifications, and filtering recipes by ingredient name, food group, or maximum calories.

## Features
- Enter new recipes
- Display all recipes sorted in alphabetical order
- Select and display a specific recipe
- Reset current recipe quantities
- Scale current recipe
- Set high-calorie notification
- Filter recipes by ingredient name, food group, or maximum calories
- Exit the application
